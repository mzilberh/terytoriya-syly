# Територія Сили — лендинг тренажерного залу (м. Броди)

Статичний сайт-лендинг. Без фреймворків, без збірки — чистий HTML + CSS + трохи
ванільного JS. Перенесено з Claude Design (`.dc.html`) у звичайний вебсайт, який
можна відкривати в будь-якому браузері, редагувати й одразу деплоїти.

## Структура

```
index.html         головна сторінка (лендинг)
styles.css         усі стилі + дизайн-токени (кольори/шрифти) + адаптив
main.js            модалки заявок, валідація форми, мобільне меню
oferta.html        Публічна оферта      (каркас — текст готує юрист)
refund.html        Повернення коштів    (каркас)
terms.html         Умови надання послуг (каркас)
images/            фото, оброблені під фірмову гаму (dark → orange)
images/_src/       вихідні кольорові фото (для повторної обробки)
tools/grade-photos.py  скрипт, що накладає фірмовий грейд на фото
```

## Фото та фірмовий грейд

Фото взято з Pexels (безкоштовно, зокрема для комерції, без атрибуції) і
оброблено під палітру: яскравість фото → тритон **#0D0D0D → #E8471C → тепле
світло**. Тому всі знімки виглядають як один сет.

- **Замінити фото:** поклади нове кольорове фото в `images/_src/` з тим самим
  ім'ям (напр. `hero.jpg`) і запусти `python tools/grade-photos.py` — воно
  перезапише оброблений файл у `images/`.
- **Підсилити/послабити грейд:** зміни список `STOPS` угорі
  `tools/grade-photos.py` (напр. зроби помаранчевий менш насиченим) і перезапусти.
- Потрібен `pip install Pillow`.

## Як подивитися

Просто відкрий `index.html` подвійним кліком у браузері. Для «як на проді»
(коректний роутинг, шрифти, iframe-мапа) — запусти локальний сервер:

```bash
# будь-що з переліку, з папки проєкту:
python -m http.server 8000        # → http://localhost:8000
npx serve                          # альтернатива
```

## Що вже працює

- Повністю адаптивна верстка (десктоп / планшет / мобільний, бургер-меню).
- 8 кнопок-заявок відкривають модалку з різними полями:
  купити абонемент (з вибором тарифу), перше тренування, пакет «Ветеранський»,
  запис до тренера, дієтолог, спортпит, заморозка, питання.
- Валідація: ім'я, український телефон (`0XXXXXXXXX` або `380XXXXXXXXX`), згода на
  обробку даних. Антиспам-honeypot.
- Після відправки — екран «Дякуємо». Дані зараз лягають у `console.log`.

## Що треба зробити далі (замовник / розробник)

1. **Підключити бекенд форми.** У `main.js` → `onSubmit()` замість `console.log`
   зробити `fetch()` на свій endpoint (Cloudflare Worker / Google Apps Script /
   тощо), який шле лист на e-mail + повідомлення в Telegram-бот і повертає
   посилання на оплату WayForPay.
2. **Реальні дані:** телефон `+380 67 000 00 00`, адреса, ФОП/РНОКПП, Instagram —
   знайти й замінити по всіх файлах (пошук: `+380 67 000 00 00`, `[Вулиця`, `[ПІБ`).
3. **Фото.** Зараз стоять стокові знімки (Pexels), оброблені під гаму. Коли
   зʼявляться реальні фото залу — поклади їх у `images/_src/` і перезапусти
   `tools/grade-photos.py` (див. розділ «Фото та фірмовий грейд»).
4. **Юридичні тексти** в `oferta.html` / `refund.html` / `terms.html` — вимога
   WayForPay для активації мерчанта.

## Дизайн-токени

Кольори і шрифти зібрані в `:root` (верх `styles.css`). Основний акцент —
помаранчевий `#E8471C`, «ветеранський» жовтий — `#F5C518`, фон — `#0D0D0D`.
Шрифти: Unbounded (заголовки), Oswald (акценти), Manrope (текст).

> Примітка: шрифти зараз тягнуться з Google Fonts CDN. За потреби (приватність /
> швидкість) можна self-host через Fontsource — див. PROGRESS.md.
